# Enterprise-Linux-Ops-Lab | Infrastructure Audit Project

A professional infrastructure health audit project focused on RHEL/Rocky Linux system administration and production-grade environment validation.

## Infrastructure Environment
* **Cloud Provider:** AWS (Amazon Web Services)
* **Service:** EC2 (Elastic Compute Cloud)
* **OS Platform:** 
    * Phase 01: RHEL 10.2 (Initial Discovery)
    * Phase 02 onwards: Rocky Linux 10 (Production-Grade, RHEL-Compatible)
* **Architecture:** Cloud-Native Instance Deployment

## Project Roles
* **Lead Infrastructure Architect:** Kunal Kashyap
* **Security & Compliance Lead:** Kanchan Kashyap

## Methodology
* **Phase 01: Infrastructure Discovery & Peer-Audit (Completed)**
    * Focus: OS integrity, Resource Capacity Analysis, and Security Compliance on RHEL 10.2.
* **Phase 02: Super-Admin Bootstrap (Completed)**
    * Focus: Automated provisioning of enterprise diagnostic toolsets on Rocky Linux 10.
    * Note: Migrated to Rocky Linux for optimized package management while maintaining full RHEL binary compatibility.

## My Contribution (Kunal Kashyap)
As the Lead Infrastructure Architect, I performed:
* End-to-end setup of RHEL 10.2 and Rocky Linux 10 instances.
* Automated system discovery and performance baseline configuration.
* Operational framework design for high-velocity environments, ensuring seamless migration between RHEL-based distributions.
